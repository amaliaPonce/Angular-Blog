# Angular Blog

This project is a simple blog built with Angular. It includes a blog list, article detail view and authentication pages.

## Requirements
- Node.js 16+
- [Angular CLI](https://angular.io/cli)

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure the backend URL in `src/environments/environment.ts` and `src/environments/environment.prod.ts`.
3. Run the development server:
   ```bash
   ng serve
   ```
   The application will be available at `http://localhost:4200/`.

## Testing
The project uses Karma for unit tests. Execute:
```bash
npm test
```

Make sure Angular CLI is installed globally if the `ng` command is missing.
